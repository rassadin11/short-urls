# О проекте

Fullstack-приложение для укорачивания ссылок, написанное на react + node.js. Ниже приведено описание API:

## API

### Endpoint: '/api/minify'
Вход: POST запрос, принимает введённый пользователем URL в формате JSON. Пример запроса:

```
const response = await fetch(`/api/minify`, {
    method: "POST",
    headers: {
        "Content-Type": "application/json",
    },
    body: JSON.stringify({ url }),
});
```

### Endpoint: '/:shortId'
Вход: GET запрос. Параметризированный URL, значение shortId зависит от ссылки. Используется, когда пользователь переходит по уже укороченной ссылке. Внутри endpointа в зависимости от того удовлетворяет ли ссылка необходимым инструкциям происходят разные действия: 

    1. переход на исходную ссылку
    2. уведомление о том, что ссылка устарела 
    3. уведомление о том, что ссылки в целом нет в базе данных

## Клиент

Клиент содержит 3 страницы:

    1. Основная страница, находящаяся в корневой ссылке
    2. Страница, уведомляющая пользователя о том, что ссылка просрочена. Находится по адресу **"/expired"**
    3. Страница, уведомляющая пользователя о том, что ссылки не существует в базе данных. Находится по адресу **"/no-page"**
